= Fonctionnement =

Le [plugin Wordpress Smart Agenda](https://go.smartagenda.fr/activation2.html?source=wordpress&utm_source=wordpress&utm_medium=link) vous permet d'intégrer la [prise de rendez-vous en ligne](https://www.smartagenda.fr/activation2.html?source=wordpress&utm_source=wordpress&utm_medium=link) directement sur votre site en quelques clics.

* Installez le plugin sur votre site Wordpress
* Ajouter [smartagenda agenda='le nom de votre agenda'] dans une page de votre choix.
(Exemple si votre agenda est https://www.smartagenda.fr/pro/demo, il faut mettre : [smartagenda agenda='demo'])
* Admirez le résultat !

== Installation ==

1. Téléchargez le module et dézippez-le dans wp-content/plugins ou installez le module via Extensions / Ajouter.
2. Activez le module dans le menu des Extensions Wordpress.
3. Ajouter le shortcode [smartagenda agenda='le nom de votre agenda'] dans une page de votre choix.
4. Admirez le résultat !
